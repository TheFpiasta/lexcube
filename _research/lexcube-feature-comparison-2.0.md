# Lexcube Feature Comparison (Jupyter vs Web)

This table lists user-facing capabilities (what you can do), with environment-specific access notes and code entry points.

| Feature                         | Description                                                                               | Jupyter                                                                                                                                                                                                                                                                                     | Web                                                                                                                                                                                                                                                                                        |
|---------------------------------|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Visualize 3D data cube          | Render a 3D volume from a 3D array and display it interactively.                          | access: `Cube3DWidget(data)` or `.plot()`; entry: `lexcube/cube3d.py` + widget bridge `src/widget.ts`; render: `src/lexcube-client/src/client/rendering.ts`; data source: `lexcube/lexcube_server/src/lexcube_widget.py`; requires WebGL2 in frontend                                       | access: standalone client page; entry: `src/lexcube-client/src/client/client.ts` -> `CubeClientContext`; render: `src/lexcube-client/src/client/rendering.ts`; data via tile server over WebSocket in `src/lexcube-client/src/client/networking.ts`; requires WebGL2/WebSocket/WebAssembly |
| Navigate camera                 | Rotate, pan, and zoom the scene to view the cube from different angles.                   | access: mouse/touch on canvas; controls in `src/lexcube-client/src/client/OrbitControls.ts` and `src/lexcube-client/src/client/interaction.ts`; same in widget mode                                                                                                                         | access: mouse/touch on canvas; controls in `src/lexcube-client/src/client/OrbitControls.ts` and `src/lexcube-client/src/client/interaction.ts`                                                                                                                                             |
| Select subranges (X/Y/Z)        | Choose a subset of the cube by adjusting range sliders for each axis.                     | access: in-widget sliders from `src/widget.ts` template + optional external sliders via `Cube3DWidget.show_sliders()` in `lexcube/cube3d.py`; programmatic control via `xlim/ylim/zlim` traitlets in `lexcube/cube3d.py`; selection logic in `src/lexcube-client/src/client/interaction.ts` | access: sliders in `src/lexcube-client/src/client/index.html` and `src/lexcube-client/src/client/interaction.ts`; updates URL fragment when not in widget mode                                                                                                                             |
| Zoom into faces                 | Zoom in/out on cube faces to change visible window size.                                  | access: mouse wheel / touch gestures; selection resize in `src/lexcube-client/src/client/interaction.ts` `changeZoomOnFace`                                                                                                                                                                 | access: mouse wheel / touch gestures; selection resize in `src/lexcube-client/src/client/interaction.ts` `changeZoomOnFace`                                                                                                                                                                |
| Animate along a dimension       | Play an animation by stepping a chosen axis through a range.                              | access: animation UI in widget template `src/widget.ts`; logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                                            | access: animation UI in `src/lexcube-client/src/client/index.html`; logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                                |
| Record animation (GIF/MP4/WebM) | Capture animation frames and export as GIF/MP4/WebM.                                      | access: animation recording checkbox + format in widget UI; implementation in `src/lexcube-client/src/client/rendering.ts` and `src/lexcube-client/src/client/interaction.ts`                                                                                                               | access: animation recording checkbox + format in `src/lexcube-client/src/client/index.html`; implementation in `src/lexcube-client/src/client/rendering.ts`                                                                                                                                |
| Export screenshot (PNG)         | Download a still image of the cube (optionally with UI overlays).                         | access: `Cube3DWidget.savefig(include_ui, dpi_scale)` in `lexcube/cube3d.py`; message handled in `src/widget.ts` -> `downloadScreenshotFromUi` in `src/lexcube-client/src/client/rendering.ts`                                                                                              | access: download button in `src/lexcube-client/src/client/index.html`; currently triggers `downloadScreenshotFromUi(true)` in `src/lexcube-client/src/client/interaction.ts`                                                                                                               |
| Export print template           | Generate a foldable paper cube template (SVG/PNG).                                        | access: UI button or `Cube3DWidget.save_print_template()` in `lexcube/cube3d.py`; render/export in `src/lexcube-client/src/client/rendering.ts` and widget handling in `src/widget.ts`                                                                                                      | access: UI button in `src/lexcube-client/src/client/index.html`; render/export in `src/lexcube-client/src/client/rendering.ts`; includes download links for SVG/PNG and note edit in UI                                                                                                    |
| Add custom note to template     | Insert a user-defined note into the print template before downloading.                    | access: “Add custom note” link in widget template; prompt + SVG injection in `src/lexcube-client/src/client/interaction.ts` around print template logic; UI element in `src/widget.ts`                                                                                                      | access: “Add custom note” link in `src/lexcube-client/src/client/index.html`; prompt + SVG injection in `src/lexcube-client/src/client/interaction.ts` `download-print-template-result-edit-note`                                                                                          |
| Choose colormap                 | Switch between built-in colormaps.                                                        | access: colormap UI in widget template `src/widget.ts`; also Python `cmap` traitlet in `lexcube/cube3d.py`; selection logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                               | access: colormap UI in `src/lexcube-client/src/client/index.html`; selection logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                       |
| Set colormap bounds             | Adjust min/max range, flip colormap, and toggle statistical auto-bounds.                  | access: UI controls + Python `vmin`/`vmax` traitlets; logic in `src/lexcube-client/src/client/interaction.ts` + `src/lexcube-client/src/client/tiledata.ts`; widget sync in `src/widget.ts`                                                                                                 | access: UI controls in `src/lexcube-client/src/client/index.html`; logic in `src/lexcube-client/src/client/interaction.ts` + `src/lexcube-client/src/client/tiledata.ts`                                                                                                                   |
| Use custom colormap data        | Provide an explicit colormap array instead of selecting by name.                          | access: pass list to `cmap` in `lexcube/cube3d.py`; conversion + selection in `src/widget.ts` `colormapChanged` and `src/lexcube-client/src/client/interaction.ts` `selectColormapByData`                                                                                                   |                                                                                                                                                                                                                                                                                            |
| Per-parameter colormap defaults | Apply parameter-specific colormap presets and bounds from metadata.                       | access: metadata lookup in `src/lexcube-client/src/client/interaction.ts`; depends on dataset metadata available to widget                                                                                                                                                                  | access: metadata lookup in `src/lexcube-client/src/client/interaction.ts`; data in `src/lexcube-client/src/content/parameterCustomColormaps.json`                                                                                                                                          |
| Color scale legend              | Show color scale with numeric labels and units.                                           | access: UI elements in `src/widget.ts` template; updates in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                                                  | access: UI elements in `src/lexcube-client/src/client/index.html`; updates in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                               |
| Hover inspect values            | Show data values and coordinates under the cursor.                                        | access: hover on cube; computed in `src/lexcube-client/src/client/interaction.ts` `updateHoverInfo`                                                                                                                                                                                         | access: hover on cube; computed in `src/lexcube-client/src/client/interaction.ts` `updateHoverInfo`                                                                                                                                                                                        |
| Axis labels and bounds          | Display axis names and min/max labels around the cube.                                    | access: overlay elements in `src/widget.ts` template; updated in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                                             | access: overlay elements in `src/lexcube-client/src/client/index.html`; updated in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                          |
| GeoJSON region borders          | Overlay region borders on the cube.                                                       | access: `Cube3DWidget.overlay_geojson()` or `overlaid_geojson` traitlet in `lexcube/cube3d.py`; validation in `lexcube/cube3d.py`; requires xarray input; render in `src/lexcube-client/src/client/rendering.ts` `loadRegionBordersFromGeoJsonForWidget`                                    | access: auto-load Natural Earth borders when geospatial context is available; `src/lexcube-client/src/client/rendering.ts` `loadRegionBordersFromNaturalEarth` + `geojson-loader.worker.ts`; no UI for custom GeoJSON                                                                      |
| Show current GPS location       | Display browser geolocation on the cube.                                                  | access: GPS button in widget UI; geolocation via `navigator.geolocation` in `src/lexcube-client/src/client/interaction.ts`; render uniform in `src/lexcube-client/src/client/rendering.ts`; requires browser permission and valid geospatial context                                        | access: GPS button in `src/lexcube-client/src/client/index.html`; geolocation via `navigator.geolocation` in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                |
| Fullscreen view                 | Toggle fullscreen mode.                                                                   | access: fullscreen button in widget template; logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                                                       | access: fullscreen button in `src/lexcube-client/src/client/index.html`; logic in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                           |
| Dataset and parameter info      | View dataset metadata and parameter attribution in the UI.                                | access: dataset info dialog in widget template; data provided by `lexcube/lexcube_server/src/lexcube_widget.py` from xarray attrs/coords; attribution parsing in `src/lexcube-client/src/client/interaction.ts`                                                                             | access: dataset info dialog in `src/lexcube-client/src/client/index.html`; metadata from `/api/datasets` and `/api/datasets/{id}`; attribution parsing in `src/lexcube-client/src/client/interaction.ts` with `parameterMetadataAttribution.json`                                          |
| Switch datasets and parameters  | Choose among multiple datasets and parameters.                                            |                                                                                                                                                                                                                                                                                             | access: dataset/parameter dropdowns in `src/lexcube-client/src/client/index.html`; metadata from `/api/datasets` in `src/lexcube-client/src/client/networking.ts`; selection in `src/lexcube-client/src/client/interaction.ts`                                                             |
| Use custom datasets             | Visualize your own data cube.                                                             | access: pass 3D `numpy.ndarray` or `xarray.DataArray` into `Cube3DWidget` in `lexcube/cube3d.py`; shape validation in `lexcube/lexcube_server/src/lexcube_widget.py`; served via widget tile server                                                                                         | access: configure tile server `lexcube/lexcube_server/config_example.json` + `lexcube/lexcube_server/src/tile_server.py`; supports local files and remote HTTP/S3 via `fsspec` in `tile_server.py` `open_dataset`                                                                          |
| Multi-dataset server features   | Manage multiple datasets, parameters, and anomalies on the server.                        |                                                                                                                                                                                                                                                                                             | access: server config `lexcube/lexcube_server/config_example.json`; implementation in `lexcube/lexcube_server/src/tile_server.py` dataset config, parameter allow/ignore lists, and `calculateYearlyAnomalies` with `_lxc_anomaly` parameters                                              |
| Deep-link selection state       | Share a URL that includes cube/parameter selection and ranges.                            |                                                                                                                                                                                                                                                                                             | access: URL fragment using `!` and `/` separators; parsed and updated in `src/lexcube-client/src/client/interaction.ts` `parseUrlFragment`/`updateUrlFragment`                                                                                                                             |
| Scripted automation             | Drive the client via browser console functions (e.g., select cube/parameter, screenshot). |                                                                                                                                                                                                                                                                                             | access: `?scripted` flag; window hooks in `src/lexcube-client/src/client/client.ts` set `window.selectCube`, `window.selectParameter`, `window.downloadScreenshotFromConsole`, etc.                                                                                                        |
| Advanced modes                  | Enable debug/expert/studio/noUi/orchestration/textureFiltering modes.                     |                                                                                                                                                                                                                                                                                             | access: URL flags in `src/lexcube-client/src/client/client.ts`: `debug`, `expert`, `studio`, `noUi`, `orchestrationMinion`, `orchestrationMaster`, `scriptedMultiView`, `textureFiltering`, `isometric`                                                                                    |
| Isometric view                  | Render the cube with isometric projection.                                                | access: `isometric_mode` traitlet in `lexcube/cube3d.py`; applied in `src/widget.ts` and `src/lexcube-client/src/client/client.ts`                                                                                                                                                          | access: `?isometric` URL flag in `src/lexcube-client/src/client/client.ts`; applied in `CubeClientContext`                                                                                                                                                                                 |
| Display quality control         | Change render quality / resolution.                                                       |                                                                                                                                                                                                                                                                                             | access: quality select in `src/lexcube-client/src/client/index.html`; applied in `src/lexcube-client/src/client/interaction.ts`                                                                                                                                                            |
| Dimension wrapping (overflow)   | Allow selection to wrap around a dimension (e.g., global longitude).                      | access: `xwrap/ywrap/zwrap` traitlets in `lexcube/cube3d.py`; applied via `src/widget.ts` -> `rendering.updateOverflowSettings`                                                                                                                                                             | access: dataset tags inferred in `src/lexcube-client/src/client/interaction.ts`; applied via `rendering.updateOverflowSettings`; no UI toggle                                                                                                                                              |
| Scale cube axes                 | Stretch or compress the cube dimensions for visual emphasis.                              | access: `cube_scale` traitlet in `lexcube/cube3d.py`; passed into `CubeClientContext` in `src/widget.ts` constructor; used in `src/lexcube-client/src/client/client.ts` and rendering pipeline                                                                                              |                                                                                                                                                                                                                                                                                            |
| Save/restore camera angle       | Reproduce exact camera position/rotation for consistent views.                            | access: `camera_angle` traitlet in `lexcube/cube3d.py`; sync in `src/widget.ts` `cameraAngleChanged` and `updateWidgetCameraAngle`                                                                                                                                                          | limited: URL `camera=<face>` selects preset face in `src/lexcube-client/src/client/interaction.ts` `applyCameraPreset`; no full numeric camera API in UI                                                                                                                                   |
| Export selected data            | Retrieve the data sub-cube for the current selection.                                     | access: `Cube3DWidget.get_current_cube_selection()` in `lexcube/cube3d.py`; supports index-only or data slicing for numpy/xarray                                                                                                                                                            |                                                                                                                                                                                                                                                                                            |
| Download progress feedback      | Show tile download progress and status messages.                                          | access: progress updates sent from widget server; `request_progress` traitlet in `lexcube/cube3d.py` and UI update in `src/widget.ts` -> `updateRequestProgressFromWidget`                                                                                                                  | access: status/tiles progress in `src/lexcube-client/src/client/interaction.ts` using download counters from `tiledata.ts`                                                                                                                                                                 |
| External slider panel           | Show separate ipywidgets sliders outside the canvas.                                      | access: `Cube3DWidget.show_sliders()` in `lexcube/cube3d.py`; implemented by `Sliders` class in same file; links to `xlim/ylim/zlim`                                                                                                                                                        |                                                                                                                                                                                                                                                                                            |
| Interactive tutorial            | Show mouse/touch tutorial overlay.                                                        |                                                                                                                                                                                                                                                                                             | access: tutorial overlay in `src/lexcube-client/src/client/index.html`; touch/mouse detection logic in inline script                                                                                                                                                                       |
| Unit conversion and formatting  | Display values in human-friendly units and precision.                                     | access: hover and legend formatting in `src/lexcube-client/src/client/interaction.ts`; unit conversion tied to parameter metadata                                                                                                                                                           | access: hover and legend formatting in `src/lexcube-client/src/client/interaction.ts`; unit conversion tied to parameter metadata                                                                                                                                                          |
